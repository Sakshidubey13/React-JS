# 💼  Developer Portfolio

A modern and responsive **Developer Portfolio Website** built using **React.js**.  
This portfolio showcases my projects, skills, and contact information as a **Full Stack MERN Developer**.

---

## ✨ Features

- ⚡ Modern UI Design
- 🎯 Responsive Layout
- 🌙 Dark Mode Toggle
- ⌨️ Typing Animation
- 🎨 Particle Background
- 📱 Mobile Friendly
- 🔗 Social Media Links
- 📂 Projects Showcase

---

## 🛠️ Tech Stack

**Frontend**

- React.js
- Tailwind CSS
- JavaScript
- HTML5
- CSS3

**Libraries**

- react-typed
- framer-motion
- lucide-react
- tsparticles
- AOS Animation
